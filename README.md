# SIGLOR
## Sistema Inteligente de Gestión Logística y Optimización de Rutas

### Proyecto final de Estructuras de Datos y Algoritmos II de la Universidad Provincial
### de Ezeiza, carrera Desarrollo de Software.
### Integrantes:

- Bravo Lautaro

SIGLOR es un Sistema Inteligente de Gestión Logística y Optimización de Rutas, teniendo
como base diferentes nodos localizados en la provincia de Buenoas Aires, su objetivo es
encontrar el camino más corto entre cualquiera de éstos. Usará teoría de grafos y algoritmos
matemáticos para realizar su tarea, y tendrá funciones como:
- Gestión de nodos
- Gestión de conexiones
- Visualización gráfica
- Gestión de contingencias
- Registro histórico
El stack principal será C++ y sus correspondientes componentes (TImage, TComboBox...).
